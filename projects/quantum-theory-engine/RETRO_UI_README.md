## ğŸ› Troubleshooting

### Colors don't appear correctly
- Make sure you're using a terminal with ANSI color support
- Try iTerm2 on macOS or modern Linux terminals

### Permission denied error

### Python not found
```bash

# Or use full path

```
