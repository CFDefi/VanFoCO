# Projects Directory

This directory contains project-specific resources, documentation, and assets for VanFossen Corporation projects.

## Structure

Each project may have its own subdirectory containing:

- Project documentation
- Design files and assets
- Configuration examples
- Additional resources

## Active Projects

See `projects.json` in the root directory for the complete list of active and planned projects.

## Adding Resources

To add resources for a project:

1. Create a directory with your project ID: `projects/your-project-id/`
2. Add your resources to that directory
3. Update the project's README.md to document the resources

Refer to `docs/CONTRIBUTING.md` for detailed guidelines.
