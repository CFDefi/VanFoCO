# VanFoCO - VanFossen Corporation

Welcome to the VanFossen Corporation project repository! This repository serves as a central hub for all company projects and their information, which are showcased on [vanfossen.us](https://vanfossen.us).

## About

VanFossen Corporation (VanFoCO) is committed to delivering innovative solutions across various technology domains. This repository contains:

- Project listings and metadata
- Project documentation and resources
- Company standards and guidelines

## Repository Structure

```
VanFoCO/
├── projects.json          # Master list of all projects with metadata
├── projects/              # Project-specific resources and documentation
├── docs/                  # Repository documentation and guidelines
└── README.md             # This file
```

## Projects

All company projects are catalogued in `projects.json`. This structured data file contains:

- Project names and descriptions
- Development status
- Technologies used
- Links to repositories and live sites
- Project categories and timelines

Visit [vanfossen.us](https://vanfossen.us) to see our projects in action.

## Adding a New Project

To add a new project to the portfolio:

1. Update `projects.json` with your project details
2. (Optional) Create a directory under `projects/` for project-specific resources
3. Submit a pull request

For detailed instructions, see [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md).

## Project Categories

- **Web Development**: Web applications and websites
- **Mobile Development**: iOS and Android applications
- **Data Science**: Analytics and machine learning projects
- **Infrastructure**: DevOps and system architecture
- **Tools**: Development utilities and frameworks
- **General**: Other innovative projects
